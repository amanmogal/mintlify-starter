# Quick Start

## Installation

1. Clone the repository:
```bash
git clone https://github.com/amanmogal/remac-app.git
cd remac-app
```

2. Open the project in Xcode:
```bash
open AppleSerialScanner.xcodeproj
```

3. Select your development team in Signing & Capabilities

4. Build and run on simulator or device

## Basic Usage

### Starting a Scan

```swift
let viewModel = SerialScannerViewModel()

// Start scanning
viewModel.startScanning()

// Enable OCR
viewModel.setOCREnabled(true)
```

### Handling Results

```swift
// Observe recognized text
viewModel.$recognizedText
    .sink { text in
        print("Detected: \(text)")
    }
```

### Camera Controls

```swift
// Toggle flash
viewModel.toggleFlash()

// Set zoom
viewModel.setZoomFactor(2.0)
```

## Configuration

### Remote Config

```swift
let config = RemoteConfigManager.shared.current
let minConfidence = config?.minConfidenceScore ?? 0.65
```

### Feature Flags

```swift
if FeatureGate.isEnabled(.enhancedOCR) {
    // Use enhanced OCR
}
```
