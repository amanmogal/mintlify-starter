# Views

## SerialScannerView

**File**: `Views/SerialScannerView.swift`

Main scanning interface.

**Components**:
- Camera preview
- ROI overlay
- Control buttons
- Guidance display
- Result display

## VisionKitLiveScannerView

**File**: `Views/VisionKitLiveScannerView.swift`

VisionKit DataScannerViewController integration.

**Features**:
- Real-time text recognition
- ROI configuration
- Temporal consistency tracking
- Pattern validation

**Configuration**:
- Quality level: `.accurate`
- ROI: `(0.1, 0.2, 0.8, 0.5)` (normalized)
- Stability: 3 consecutive frames
- Throttle: 100ms

## ScanningGuidanceView

**File**: `Views/ScanningGuidanceView.swift`

User guidance display.

**Features**:
- Real-time guidance messages
- Animated cards
- Status indicators

## CharacterConfidenceView

**File**: `Views/CharacterConfidenceView.swift`

Confidence visualization.

**Features**:
