# Configuration

## RemoteConfig

Configuration is managed via `RemoteConfigManager`.

### Key Settings

<ParamField path="minConfidenceScore" type="Float" default="0.65">
  Minimum confidence threshold for serial detection.
</ParamField>

<ParamField path="requiredConsecutive" type="Int" default="2">
  Number of consecutive frames required for stability.
</ParamField>

<ParamField path="roiCoordinates" type="CGRect">
  Region of interest for OCR (normalized coordinates).
</ParamField>

<ParamField path="throttleInterval" type="TimeInterval" default="0.1">
  Frame processing throttle interval in seconds.
</ParamField>

### Access

```swift
let config = RemoteConfigManager.shared.current
let minConfidence = config?.minConfidenceScore ?? 0.65
```
